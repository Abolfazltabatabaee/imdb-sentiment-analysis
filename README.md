# IMDb Sentiment Analysis

This project performs sentiment analysis on IMDb movie reviews using classical machine learning models.

The goal is to classify movie reviews as **positive** or **negative** based on their text content.

---

## Overview

- Dataset: IMDb Movie Reviews
- Task: Binary sentiment classification (positive / negative)
- Feature extraction:
  - Bag of Words
  - TF-IDF
- Models:
  - Naive Bayes
  - Multi-layer Perceptron (MLP)
- Evaluation:
  - Accuracy
  - Precision
  - Recall
  - Confusion Matrix (2Ã—2, printed in terminal)
